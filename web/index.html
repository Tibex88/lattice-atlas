Specification

Program Definitions

This specification defines the multilattices, residuated lattices, residuated multilattices, homomorphisms, and derivations implemented by the program. The residuated lattices are bounded and commutative, with top as the multiplicative unit.

1. Multilattices

For a poset (P, ≤) and any subset X ⊆ P, the upper bounds are X↑ = {p ∈ P | p ≥ x for all x ∈ X} and the lower bounds are X↓ = {p ∈ P | p ≤ x for all x ∈ X}.

Ordered Multilattice

A poset (M, ≤) is an ordered multilattice when every element of {x, y}↑ lies above an element of x ⊔ y, and every element of {x, y}↓ lies below an element of x ⊓ y.

Multilattice

The algebra (M, ⊔, ⊓) associated with an ordered multilattice (M, ≤).

Bounded Multilattice

The algebra (M, ⊔, ⊓, 0, 1) associated with an ordered multilattice having least element 0 and greatest element 1.

Homomorphism

A function f : M1 → M2 satisfying f(x ⊔ y) ⊆ f(x) ⊔ f(y) and f(x ⊓ y) ⊆ f(x) ⊓ f(y).

Derivation

Given endomorphisms f and g, an (f, g)-derivation d satisfies d(x ⊓ y) ⊆ (d(x) ⊓ f(y)) ⊔ (g(x) ⊓ d(y)).

2. Residuated Lattices

Residuated Lattice

An algebra (L, ∨, ∧, ⊙, →, 0, 1) where (L, ∨, ∧, 0, 1) is a bounded lattice, (L, ⊙, 1) is a commutative monoid, and → satisfies the adjointness law x ⊙ z ≤ y ⇔ x ≤ z → y.

Homomorphism

A map preserving join, meet, multiplication, implication, and the constants 0 and 1.

Multiplicative Derivation

An (f, g)-multiplicative derivation satisfies d(x ⊙ y) = (d(x) ⊙ f(y)) ∨ (g(x) ⊙ d(y)).

Implicative Derivation

An (f, g)-implicative derivation satisfies d(x → y) = (d(x) → f(y)) ∨ (g(x) → d(y)).

3. Residuated Multilattices

Residuated Multilattice

An algebra (M, ⊔, ⊓, ⊙, →, 0, 1) where (M, ⊔, ⊓, 0, 1) is a bounded multilattice, (M, ⊙, 1) is a commutative monoid, and adjointness holds: x ⊙ z ≤ y ⇔ x ≤ z → y.

Homomorphism

A map preserving multi-meet, multi-join, multiplication, implication, and the constants 0 and 1, with the meet and join conditions using subset inclusion.

Multiplicative Derivation

An (f, g)-multiplicative derivation satisfies d(x ⊙ y) ∈ (d(x) ⊙ f(y)) ⊔ (g(x) ⊙ d(y)).

Implicative Derivation

An (f, g)-implicative derivation satisfies d(x → y) ∈ (d(x) → f(y)) ⊔ (g(x) → d(y)).
