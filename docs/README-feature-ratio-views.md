# Feature: Ratio and Percentage Views

## Goal
Translate raw counts into proportions that are easier to interpret.

## What It Should Show
- Percent of lattices that admit residuated expansions
- Percent of residuated lattices that are linear
- Percent of structures with selected properties

## Why It Matters
The paper's conclusions are not only about totals but also about rarity and prevalence.

## Suggested UI
- Percentage cards
- Side-by-side comparisons
- Toggle between raw counts and percentages

## UI Shape
- Small summary cards with bold percentages
- Inline ratio labels beside count charts and tables
- A mode toggle that switches the same component between counts and percentages

## Data Needed
- Aggregate counts
- Property counts

## Status
Not implemented.
