# Feature: Smallest-Example Finder

## Goal
Find the least-size example of a property or property combination.

## What It Should Show
- Smallest BL-algebra
- Smallest lattice with a selected property combination
- Smallest residuated lattice matching constraints

## Why It Matters
The paper repeatedly highlights least examples of rare combinations.

## Suggested UI
- Search form for property sets
- Result card with size, graph, and tables

## UI Shape
- A compact query form asking for one or more properties
- A result card showing the least-size match
- Quick actions to open the result in the compare panes

## Data Needed
- Property evaluations across all sizes

## Status
Completed for the current milestone.
