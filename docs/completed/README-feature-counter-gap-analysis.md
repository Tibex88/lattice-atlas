# Feature: Counter-Gap Analysis

## Goal
Use the residuum and order structure to identify sparse or missing counter-actions in a chosen blueprint.

## What It Should Show
- Candidate gaps in the current shape
- Suggested strongest missing counter regions
- Explanations of why a region is sparse

## Why It Matters
This is one of the most direct ways to turn the algebra into actionable content-planning guidance.

## Suggested UI
- Gap report panel
- Highlighted node pairs or regions
- Ranked suggestions

## UI Shape
- A report section under the selected blueprint
- Ranked rows describing a missing or weak counter region
- Click-through actions that focus the relevant nodes in the structure view

## Data Needed
- Residuated structures with decoded `→`
- Derived heuristics for sparse capability regions

## Status
Completed for the current milestone.
