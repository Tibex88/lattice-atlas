# Feature: Why-Qualified Explanations

## Goal
Explain why a structure satisfies or fails a property.

## What It Should Show
- Short explanations
- Witnesses or counterexamples where possible

## Why It Matters
Plain true/false labels help browsing, but explanations help understanding.

## Suggested UI
- Expandable notes below each property
- Inline examples for failures

## Data Needed
- Property-specific explanation logic

## Status
Not implemented.
