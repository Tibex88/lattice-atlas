# Priority Feature List

This file explains the first implementation slice recommended for the site.

## Status
The original first-slice items below are completed in the current milestone.

## Related Follow-On Work
- [0bc338a](https://github.com/Tibex88/lattice-atlas/commit/0bc338a) adds the next research tools beyond the first slice: property checker, derived operations, co-occurrence matrix, constraint URL state, and export controls.

## 1. Property Filters and Badges
Highest value because the paper is strongly organized around structural and algebraic properties. This unlocks meaningful browsing immediately.

## 2. Width and Height Distribution Charts
High value because width/height are major structural statistics in the paper and can be shown clearly in charts.

## 3. extlat Expansion-Count Ranking
High value because `extlat` is unique to this dataset and ranking by expansion count makes it easy to understand what the count dictionary means.

## 4. Same-Lattice Comparison Across Multiple reslat Expansions
High explanatory value because it shows how one base lattice can support multiple multiplication tables.

## 5. Appendix-Style Statistics Tables
Important for paper alignment and verification, but slightly less interactive than the features above, so it comes after the main browsing tools.

## Why This Is Not Exhaustive
This is a build-order recommendation, not a full product specification. It leaves out other useful features such as ratio views, residuum display, property explanations, smallest-example search, constraint search, and export.
